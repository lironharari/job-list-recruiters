# Job List Recruiters

Job List Recruiters is a full-stack web application for job boards and recruiter management. It allows recruiters and admins to post jobs, manage applications, communicate with candidates, and handle email templates. Applicants can browse jobs and apply directly with their resume.

## Technologies Used

- **Frontend:**
	- React
	- TypeScript
	- Vite
	- Material-UI
	- React Router
	- Context API
	- Jest
	- ESLint

- **Backend:**
	- Node.js
	- Express
	- TypeScript
	- MongoDB
	- Resend (email service)
	- Jest
	- Dotenv

## Features

- Job listing, search, and application
- Recruiter/admin dashboard
- Email template management
- Authentication and role-based access
- RESTful API endpoints
- Responsive design with light/dark mode

## Structure

- `frontend/` – React app for UI and client logic
- `backend/` – Node.js/Express API server

See the frontend and backend README files for more details and setup instructions.
