
# Job List Recruiters Frontend

This is the frontend for the Job List Recruiters project, a modern job board and recruiter management platform. It allows recruiters and admins to post jobs, manage applications, and communicate with candidates. Applicants can browse jobs and apply directly with their resume.

## Technologies Used

- **React** – UI library for building interactive user interfaces
- **TypeScript** – Type safety for React and API calls
- **Vite** – Fast development/build tool
- **Tailwind CSS** – Utility-first CSS framework for styling and dark mode support
- **React Router** – Routing and navigation
- **Context API** – Authentication and global state management
- **Jest** – Testing framework
- **ESLint** – Linting and code quality

## Features

- Job listing and search
- Job application with resume upload
- Recruiter/admin dashboard
- Email template management
- Authentication and role-based access
- Responsive design with light/dark mode

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/` – Main source code
- `components/` – Reusable UI components
- `pages/` – Application pages
- `context/` – React Context for authentication
- `api/` – API calls to backend
- `utils/` – Utility functions
